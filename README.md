# 🧠 C++ OOP Practice Programs

A collection of C++ programs written to practice Object-Oriented Programming concepts.  

---

## 📚 Concepts Covered

- Classes & Objects
- Constructors & Destructors
- Encapsulation (access specifiers, getters/setters)
- Inheritance (single, multilevel, multiple)
- Polymorphism (function/operator overloading, virtual functions)
- Abstraction & Abstract Classes
- Friend Functions
- Static Members
- Templates

---

## 🔧 How to Compile & Run
```bash
g++ filename.cpp -o output && ./output
```

---

## 🎯 Purpose

Practice repo. Written to learn C++ OOP from scratch — nothing more, nothing less.

---

## 🛠️ Requirements

- C++11 or later
- `g++` or any standard C++ compiler
